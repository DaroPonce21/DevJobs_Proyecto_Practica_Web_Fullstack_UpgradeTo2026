# DevJobs — Proyecto de Práctica desarrollador web (Upgrade To 2026)

## Descripcion
DevJobs es una web inspirada en bolsas de empleo para desarrolladores. 
El objetivo del proyecto es practicar:
- HTML semántico
- CSS moderno (Grid, Flex, variables, responsive design)
- Buenas prácticas de diseño UI
- JavaScript vanila
- React
- Hooks
- Estado Global y React Router
- Comunicación con el Servidor
- Programación de Servidores con Node.js y Express
- Testing
- Inteligencia Artificial
- TypeScript
- SQL
- CI/CD
- DevOps y Docker
